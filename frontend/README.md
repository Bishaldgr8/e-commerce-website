# Frontend - The Secret Shop

React + TypeScript frontend for The Secret Shop e-commerce platform.

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Tech Stack

- React 18
- TypeScript
- Vite
- React Router DOM
- Axios
- Lucide React Icons
- React Hook Form + Zod
- Sonner (Toast Notifications)

## API Configuration

The frontend connects to the backend API at `http://localhost:5000/api/v1` by default.

To change this, update `src/api/client.ts`.
